What's Linux???

Linux is a free and open-source kernel, inspired by UNIX but not derived from it.

When people say "Linux," they often mean "GNU/Linux," which combines the Linux kernel with GNU software.

The GNU Project, which is also free and open-source, provides many essential tools used in Linux distributions.

Most Linux distributions use the GNU userland alongside the Linux kernel to create a complete operating system.
